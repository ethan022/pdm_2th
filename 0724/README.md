## 2. 오늘의 핵심 한 줄

> **문자열 메서드는 원본을 바꾸지 않는다. 결과를 반드시 다시 받아야 한다.**

```python
s = "python"
s.upper()          # 결과가 그냥 버려짐
print(s)           # python  ← 안 바뀜!

s = s.upper()      # ← 이 패턴을 통째로 외우기
print(s)           # PYTHON
```

"분명 바꿨는데 값이 그대로다" 싶으면 **가장 먼저 이걸 확인**하세요.

## 3. 메서드와 점(.) 문법

값 하나하나가 **객체** = 값(데이터) + 그 값에 딸린 기능.
자료형은 '객체의 종류'라서, 종류마다 쓸 수 있는 기능이 다릅니다.
(문자열엔 `upper()` 가 있지만, 숫자엔 없습니다)

**메서드** = 특정 자료형에 소속된 함수.

```python
값.메서드이름()          # 점(.)을 찍어 부른다
word.upper()            # 괄호를 꼭 붙여야 실행된다
word.count("a")         # 추가 정보가 필요하면 괄호 안에
```

`word.upper` 처럼 **괄호를 빼면 실행되지 않습니다.** 흔한 실수입니다.

## 4. 한 장 치트시트

### 대소문자

| 메서드 | 하는 일 | 예 |
|---|---|---|
| `upper()` | 모두 대문자 | `"python"` → `PYTHON` |
| `lower()` | 모두 소문자 **(가장 자주 씀)** | `"NORMAL"` → `normal` |
| `capitalize()` | 문장 첫 글자만 대문자 | `"python is"` → `Python is` |
| `title()` | 단어마다 첫 글자 대문자 | `"kim chul soo"` → `Kim Chul Soo` |
| `isupper()` | 모두 대문자인지 **검사** | `"ABC"` → `True` |
| `islower()` | 모두 소문자인지 **검사** | `"abc"` → `True` |

한글·숫자·기호는 그대로 둡니다. (`"정상 normal".upper()` → `정상 NORMAL`)

**함정 2가지**

```python
"PYTHON IS FUN".capitalize()   # 'Python is fun'  ← 나머지를 소문자로 내린다
"it's normal".title()          # "It'S Normal"    ← 기호 뒤도 단어 시작으로 봄
```

### 공백

| 메서드 | 하는 일 | 예 (`s = "  대기  "`) |
|---|---|---|
| `strip()` | 앞뒤 공백 제거 **(가장 자주 씀)** | `[대기]` |
| `lstrip()` | 왼쪽(앞)만 | `[대기  ]` |
| `rstrip()` | 오른쪽(뒤)만 | `[  대기]` |
| `strip("문자")` | 앞뒤에서 그 글자를 제거 | `"###정상###".strip("#")` → `정상` |

- 공백은 눈에 안 보이니 **`"[" + s + "]"` 로 감싸서 확인**하는 습관을 들이세요.
- `strip("#$")` 처럼 여러 글자를 넣으면 **그 중 아무거나** 양 끝에서 뗍니다.
- **`strip` 은 앞뒤 전용** — 가운데 공백은 못 지웁니다.
  `"  정 상  ".strip()` → `"정 상"` (중간 공백은 다음 시간 `replace` 로)

### 체이닝

메서드 뒤에 또 메서드를 점으로 이어 붙일 수 있습니다. **왼쪽에서 오른쪽 순서**로 읽습니다.

```python
raw = "  NORMAL  "
raw.strip().lower()      # 'normal'   공백 제거 → 소문자
```

**입력값이 들어오면 일단 `strip()` → `lower()` 로 정리부터.** 표준처럼 쓰는 패턴입니다.

## 5. 왜 정리를 해야 하나

같은 상태인데 표기만 다른 데이터가 한 파일에 섞여 들어옵니다.

```
'NORMAL'   'Normal'   'normal'   '  normal  '
```

정리하지 않으면 컴퓨터는 이를 **서로 다른 네 가지 상태로 보고 따로 집계**합니다.

```python
"Fault" == "FAULT"                  # False
"Fault".lower() == "FAULT".lower()  # True   ← 통일 후 비교

"정상" == "정상 "                    # False  (뒤 공백 하나 차이)
"정상" == "정상 ".strip()            # True
```

파일명 점검도 마찬가지입니다.

```python
fname = "Sensor_LOG.CSV"
fname.endswith(".csv")              # False  (대문자 .CSV)
fname.lower().endswith(".csv")      # True
```
