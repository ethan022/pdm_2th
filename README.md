# pdm_2th

**설비 고장을 예측하는 AI 데이터 분석** 과정 학습 저장소입니다.
`pdm` = Predictive Maintenance(예측 정비).

센서 데이터로 설비의 고장 징후를 미리 읽어내는 것이 목표이고,
Python 기초 → 데이터 처리(pandas·numpy) → 시각화 → 시계열 분석 → 머신러닝 이상 탐지 순으로 진행합니다.

## 실행 방법

Python 3 설치 후, 파일이 있는 폴더에서:

```bash
python 02_01_변수.py
```

VSCode에서는 파일을 열고 **F5** 또는 오른쪽 위 **▶ 실행** 버튼.
코랩(Colab)을 쓴다면 코드를 셀에 붙여넣고 **Shift + Enter**.

복습 코드는 코드 옆에 `# →` 로 실행 결과를 적어두었습니다.
**결과를 먼저 예상한 뒤 실행해서 맞춰보는 방식**으로 사용하세요.

## Git 기본 명령어

```bash
git status                     # 지금 상태 확인
git add .                      # 변경사항 스테이징
git commit -m "메시지"          # 스냅샷 저장
git push                       # GitHub에 업로드
git pull                       # GitHub의 변경사항 가져오기

```

커밋 메시지는 **무엇을 했는지** 드러나게 씁니다. (`수정` ❌ → `문자열 슬라이싱 예제 추가` ⭕)
